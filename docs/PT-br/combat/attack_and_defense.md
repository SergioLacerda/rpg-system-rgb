# Combate

## Iniciativa

Mais pontos Verde (G) → age primeiro.
Ataque surpresa ignora iniciativa.

## Ataque

Postura ofensiva. Atacante usa pontos vermelho

Um ataque acerta quando:

```text
**R atacante ≥ G defensor**
```

Resolução:

```text
Se Agressor (R + modificadores) ≥ Alvo (G + modificadores) → acerto; caso contrário → esquiva.
```

## Defesa

Maximiza evasão. Defensor usa pontos verde.

```text
Defesa = **Armadura + Escudo**
```

## Agarrar

Permite imobilizar o oponente ou movê-lo forçadamente durante um turno.

```text
A ação de agarrar terá êxito com a seguinte resolução:
- Agressor(G) ≥ Alvo(G)
- Agressor(R) ≥ Alvo(R)

Agressor pode usar meta-habilidades.
Alvo pode usar bônus pré-declarados.
```

## Dano

DanoFinal = Agressor (R + modificadores) – Alvo (Armadura + modificadores)
VidaRestante = VidaAtual – DanoFinal

## Resolução de Dano

1. verificar acerto
2. calcular dano
3. aplicar penetração
4. reduzir armadura
5. reduzir escudo
6. aplicar vida

## Meta habilidades ou magia

### Ataque Magico durante o combate

```text
A cada turno os pontos sao recuperados para usar magias ou meta-habilidades.

**Regeneração = 1 ponto B por nível**
```

### Defesa Magica durante o combate

```text
Defesa = **Armadura + Escudo fisico (se houver) + Escudo energético (ou mágico, dependendo do cenário)**
```
