# Movimento

O movimento no **Sistema RGB** é governado principalmente pelo **vetor Verde (G)**, que representa agilidade, mobilidade e velocidade de reação.

As regras de movimento se conectam diretamente ao **Triângulo Tático RGB**, onde cada vetor representa uma estratégia diferente de combate.

```text
        G
   mobilidade / posicionamento

R ---------------- B
poder / dano        escudo / defesa
```

- **R (Vermelho)** foca em dano e poder ofensivo
- **G (Verde)** foca em mobilidade e posicionamento
- **B (Azul)** foca em defesa e resistência

O movimento, portanto, representa o **componente de vantagem posicional** no sistema RGB.

## Distância de Movimento

A distância que um personagem pode se mover durante um turno de combate é:

```text
Movimento = G × 2 metros
```

Exemplo:

```text
G = 3
Movimento = 6 metros por turno
```

Valores mais altos de **G** permitem que personagens se reposicionem mais rapidamente e controlem melhor o campo de batalha.

## Grade Tática

O Sistema RGB suporta combate tático usando uma grade quadrada.

Grade padrão:

```text
1 quadrado = 1 metro
```

Isso permite posicionamento preciso para:

- cobertura
- flanqueamento
- linha de visão
- efeitos de área

## Movimento Aéreo

Se um personagem puder voar, o movimento aéreo utiliza a mesma fórmula:

```text
Movimento Aéreo = G × 2 metros
```

O Mestre pode modificar esse valor dependendo do cenário.

Exemplos incluem:

- jetpacks
- voo mágico
- superpoderes

## Estrutura do Turno de Combate

Cada turno de combate normalmente inclui:

- **Movimento**
- **Ação Principal**
- **Ação Parcial**

Exemplos de **Ações Principais**:

- atacar
- defender
- usar habilidade

Exemplos de **Ações Parciais**:

- recarregar arma
- pequeno reposicionamento
- interagir com o ambiente

## Efeitos de Terreno

O terreno pode reduzir o movimento.

```text
Terreno     Efeito
----------  -------------
Leve        −1 movimento
Moderado    movimento ÷2
Difícil     movimento ÷3
```

Penalidades de terreno são aplicadas **após o cálculo do movimento base**.

Exemplo:

```text
G = 4
Movimento Base = 8 metros

Terreno moderado → 4 metros
```

## Regra Opcional: Corrida

Personagens podem realizar uma corrida para se mover mais rapidamente.

```text
Corrida = G × 3 metros
```

Restrições:

- Não é possível atacar no mesmo turno
- Apenas movimento e ações menores são permitidas

Essa regra permite que personagens com alta mobilidade se reposicionem rapidamente.

## Interpretação Tática

Os vetores RGB influenciam as decisões em combate:

```text
Ataque → R
Mover → G
Defender → B
```

Isso cria três estratégias distintas de combate:

```text
Estratégia           Vetor
-------------------  ------
Atacante pesado      R
Combatente móvel     G
Guardião defensivo   B
```

O movimento representa, portanto, o **controle tático do espaço** no sistema de combate RGB.
